# 翻訳記事レビュータスク

指定された翻訳記事をレビューし、問題があれば修正する。

## 入力

- ファイル: {{FILE}}

## レビュー観点

### 1. 用語の一貫性

- 技術用語が記事内で統一されているか
- 新しい技術用語に英語併記があるか（初出時）

### 2. 表記ルール

- 数式が LaTeX 記法 `$...$` になっているか
- 全角・半角間にスペースがあるか
- カッコが半角になっているか

### 3. 日本語の自然さ

- 直訳調で不自然な表現がないか
- 冗長な表現がないか（例: 「〜性能」が不要な場合）

### 4. 避けるべき表現

| 避ける表現 | 推奨 |
| --- | --- |
| 悪いイベント | 望ましくないケース |
| ブックキーピング | 整合性管理 |
| 0 中心の | 平均 0 の |

## 処理

1. ファイルを読み込み
2. 上記観点でチェック
3. 問題があれば修正を適用
4. 修正内容のサマリを出力
