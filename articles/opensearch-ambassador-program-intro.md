OpenSearch Project は、プロジェクトの成長を支えてきた開発者、メンテナー、そしてアドボケイトの皆さんの時間、アイデア、エネルギーなしには今日の姿にはなりませんでした。そこで私たちは、OpenSearch Software Foundation [アンバサダープログラム](https://opensearch.org/ambassadors/)を立ち上げることになりました。これは、期待を超える活躍をしているコミュニティメンバーを表彰し、サポートするための新しい取り組みです。

本日、初代アンバサダーを発表できることを嬉しく思います。このグループは、コントリビューター、コンテンツクリエイター、オーガナイザー、テクニカルリーダーなど、OpenSearch の学習、構築、コミュニティとのつながりを積極的に支援してきた多様なメンバーで構成されています。

アンバサダープログラムは、すでに OpenSearch コミュニティに影響を与えている人々にスポットライトを当て、彼らが得意とする活動を継続できるよう、より多くの認知とサポートを提供することを目的としています。

## 初代 OpenSearch Software Foundation アンバサダーの紹介

この最初のコホートには、世界中のさまざまな業界で働き、異なる言語 (人間の言語もプログラミング言語も!) を使い、プロジェクトに多様な視点をもたらすメンバーが含まれています。彼らは全員、OpenSearch コミュニティを健全で、インクルーシブで、技術的に強固な形で成長させることに尽力してきました。

### Amanda Katona

OpenSearch Foundation の成長と、世界中での OpenSearch 普及拡大に注力しています。生まれながらのコミュニティビルダーであり、ユーザーや組織が検索と AI を最大限に活用できるプログラムを作成しています。コペンハーゲンのダイニングからユタのトレイルまで、あらゆる場所からインスピレーションを得る冒険家でもあります。

### Charlie Hull

OpenSearch を含むオープンソースプラットフォームに深い経験を持つ検索コンサルタントです。組織が効果的な検索システムを構築するのを支援し、AI を検索に統合して結果を改善しています。率直な視点で知られ、講演者、ライター、トレーナーとしても頻繁に活動しています。

### Dotan Horovits

AWS のシニアデベロッパーアドボケイトであり、長年の OpenSearch チャンピオンです。プロジェクトの発足当初から関わり、ファウンデーション主導のモデルを推進し、現在は Observability Technical Advisory Group をリードしています。オブザーバビリティに取り組んでいないときは、OpenObservability Talks ポッドキャストのホストを務めています。

### Eric Pugh

OpenSource Connections の共同創設者であり、OpenSearch のドキュメントとレレバンス機能のコントリビューターです。組織の検索成熟度向上と効果的な OpenSearch チームの構築を支援しています。検索に集中していないときは、手作りで家具を製作しており、現在はデンマークモダンスタイルのソファを制作中です。

### Itamar Syn-Hershko

BigData Boutique の創設者であり、組織がミッションクリティカルな OpenSearch クラスターを運用するのを支援しています。検索分野で 10 年以上の経験を持ち、大規模な検索・分析システムのスケーリングについてアドバイスしています。コンサルティングや講演以外の時間は、OpenSearch コミュニティに直接貢献しています。

### Kassian Rosner Wren

OpenSearch、AI、開発者教育を専門とするオープンソースエバンジェリストです。検索と機械学習をより身近にするチュートリアル、ワークショップ、講演を作成しています。ユーモアとロボティクスへの愛情を持ち、OpenSearch コミュニティにエネルギーと創造性をもたらしています。

### Kris Freedain

OpenSearch Project とその Technical Steering Committee のコミュニティマネージャーです。グローバルなユーザーグループを構築し、コントリビューターをつなぎ、コミュニティを歓迎的な場所にしています。OpenSearch 以外では、Fosstodon Fediverse コミュニティの運営も手伝っています。

### Laysa Uchoa

分散システム、データモデリング、クラウドインフラストラクチャの専門知識を OpenSearch に提供するバックエンドエンジニアです。パフォーマンスとクリーンな設計に注力し、プロジェクトをよりスケーラブルで信頼性の高いものにしています。熱心な検索愛好家であり、システム設計について語り合う時間も大切にしています。

### Nate Boot

DevOps と Linux のバックグラウンドを持つ OpenSearch のデベロッパーアドボケイトです。AWS で約 10 年間勤務し、ユーザーグループ、カンファレンス、コミュニティサポートを通じてユーザーの OpenSearch 導入を支援しています。レトロテクノロジーにも興味があるので、マルガリータを片手に彼を見かけたら、古いビデオゲームについて聞いてみてください。

### Nils Bandener

フリーランスのソフトウェアアーキテクトであり、OpenSearch セキュリティプラグインの共同メンテナーです。ユーザビリティを念頭に置きながら、プロジェクトのセキュリティ強化を支援しています。実用的なアプローチで知られ、彼の口癖は「一歩引いて考えてみましょう」です。

### Sakshi Nasha

Cohesity のソフトウェアエンジニアであり、OpenSearch のコントリビューターです。国際カンファレンスで知識を共有し、開発者が検索技術をより活用できるよう支援しています。仕事以外では、トレッキングやサイクリングを楽しむ熱心なアスリートです。

## 今後の予定

OpenSearch で素晴らしいものを構築している方、または他の人がそうするのを支援している方は、次期アンバサダーへの[応募](https://opensearch.org/ambassadors/apply)をお待ちしています。

OpenSearch ブログやソーシャルチャンネルでは、今後数週間にわたってアンバサダーのストーリー、プロジェクト、貢献を紹介していく予定です。

OpenSearch をここまで成長させてくれたすべての皆さんに感謝します。私たちはまだ始まったばかりです。
